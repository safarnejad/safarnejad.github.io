جلسۀ هجدهم (شنبه ۲ آذر ۹۸)


  • مسئلۀ پرسش کمینۀ یک محدوده
    • برای یک لیست به طول n داده شده و هر بار کمینۀ یک محدوده از لیست مورد پرسش (query) قرار می‌گیرد.
    • ۱. راه حل ساده: هر دفعه کل محدوده را بررسی می‌کنیم.
      • پیش‌پردازش O(1) / حافظۀ اضافه O(1) / پاسخ به پرسش O(n)
    • ۲. ایدۀ جذر: لیست را به زیرقسمت‌های n√تایی افراز می‌کنیم و کمینۀ هر قسمت را در پیش‌پردازش حساب می‌کنیم. برای پاسخ به پرسش‌ها از این اطلاعات کمک می‌گیریم.
      • توضیح تصویری
      • پیش‌پردازش O(n) / حافظۀ اضافه O(√n) / پاسخ به پرسش O(√n)
    • ۳. ایدۀ توان‌های ۲: برای هر عنصر قسمت‌هایی که با آن عنصر شروع می‌شوند و طول 2i دارند را درنظر می‌گیریم. کمینۀ هر کدام از این قسمت‌ها را در پیش‌پردازش حساب می‌کنیم و برای پاسخ به پرسش‌ها از آن‌ها کمک می‌گیریم.
      • پیش‌پردازش: min(a[j, j+2i-1]) = min(min(a[j, j+2i-1-1]), min(a[j+2i-1, (j+2i-1)+2i-1-1]))
      • پاسخ به پرسش: min(a[j, k]) = min(min(a[j, j+2i-1]), min(a[k-2i+1, k]))
      • پیش‌پردازش O(n.log n) / حافظۀ اضافه O(n.log n) / پاسخ به پرسش O(1)
    • ۴. استفاده از درخت پاره‌خطی - در جلسات بعد گفته می‌شود.
      • پیش‌پردازش O(n) / حافظۀ اضافه O(n) / پاسخ به پرسش O(log n)
    • ۵. راه‌حل بهینه - در این درس گفته نمی‌شود.
      • پیش‌پردازش O(n) / حافظۀ اضافه O(n) / پاسخ به پرسش O(1)
  • درخت دکارتی
    • تعریف درخت دکارتی
    • مثال برای لیست [9, 3, 7, 1, 8, 12, 10, 20, 15, 18, 5]
    • ساخت درخت دکارتی در زمان خطی با استفاده از پشته
